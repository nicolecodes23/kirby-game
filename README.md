# Kirby Running Game

## Game Instructions 
Press enter to start.

Use arrow keys to jump up, move left and move right.

Press spacebar to restart game.

## Features
- Player Movement
- Canyon Obstacles
- Collectable Coins
- Collectable Score
- Side scrolling
- Player Life
- Sound Effects
- Particle Effect (Volcano Eruption)
- Enemies
- Platforms
